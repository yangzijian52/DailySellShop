# 更新日志

本项目遵循语义化版本号。日期使用 `YYYY-MM-DD` 格式。

## 2.0.0 - 2026-07-11

### 新增

- 新增 `/dss` 玩家购买商店，提供 10 个创造模式分类、分页、搜索、数量选择和二次确认。
- Java 版使用箱子 GUI，Floodgate 基岩版使用功能对应的表单。
- 新增 20 种最高等级普通药水、43 本最高等级附魔书和 88 种独立配置的刷怪蛋。
- 新增自定义购买数量、60 秒超时、移动/换世界/死亡/退出取消和一次错误重试。
- 新增独立购买日志、购买成功/失败音效和可配置菜单命令按钮。

### 调整

- 目标平台升级为 Paper 26.2 和 Java 25。
- 原 `config.yml` 更名为 `SellShopconfig.yml`，购买商店使用独立 `shopconfig.yml`。
- 商品池扩展为 1477 个可交易条目，并按照 Mojang 26.2 创造模式顺序分类。
- 普通购买价默认等于收购价；价格按基础资源、原版配方与稀有度重新平衡。
- 两份玩家配置补充完整中文注释，菜单标题、大小、图标、槽位、Lore、按钮和音效均可配置。

### 修复

- 修复商品分类错误及同类型商品排列混乱。
- 修复药水和附魔书无法按真实元数据识别的问题。
- 修复禁用分类仍可通过搜索访问商品的问题。
- 修复 `/ds` 经济入账失败时物品未恢复的问题。
- 修复配置批量生成时可能出现的中文问号和 UTF-8 编码问题。

### 安全

- 购买流程增加余额、权限、库存容量、商品有效性和重复提交检查。
- 发放失败时恢复背包并退款，背包空间不足时不会把物品掉落到地面。
- Java 菜单拦截点击、Shift 点击、数字键、双击和拖拽等取物操作。

### 验证

- 已在 Paper 26.2 服务器完成插件加载、Java 菜单、收购和购买交易流程实测。
