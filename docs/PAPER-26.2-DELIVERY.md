# DailySellShop Paper 26.2 交付说明

## 版本信息

- 插件版本：`2.0.0`
- Paper API：`26.2.build.53-alpha`
- Java：`25+`
- 交付日期：`2026-07-11`

## 交付内容

- 保留 `/ds` Java 箱子 GUI 与 Floodgate 基岩表单双端收购功能
- 原 `config.yml` 更名为 `SellShopconfig.yml`，首次升级自动复制旧配置
- 新增独立 `shopconfig.yml` 和 `/dss` 长期购买商店
- 商店读取 1477 项可交易商品，默认购买价等于收购价
- 包含 20 种最高等级普通药水和 43 本最高等级附魔书，收购价与购买价统一为 10
- 基础资源按获取难度定价，975 类可合成物按 Mojang 26.2 最低合法配方成本推导，稀有战利品采用来源稀有度保底价
- 矿石价格计入时运产出风险；可逆压缩原料按配方数量保持价格对应，例如红石块价格为红石粉的 9 倍
- 按 10 类自动归档商品，并支持配置覆盖单个物品的分类和售价
- 分类和顺序使用 Mojang 26.2 创造模式目录，同类型物品按原版次序连续展示
- Java 与基岩版均支持分类、分页、搜索、数量选择、二次确认和自定义数量
- 自定义输入支持 60 秒超时、2 格移动取消、切换世界、退出、死亡取消和一次重试
- 交易执行余额、权限、最新价格和背包空间校验；空间不足时按可容纳数量成交
- 使用 `BigDecimal` 计算并在扣款后发放失败时恢复背包、自动退款
- 独立日志记录时间、UUID、玩家名、Material、数量、单价和总价
- 支持配置购买成功、购买失败音效以及购买菜单自定义命令按钮
- 两份玩家配置提供完整中文注释，构建时仅过滤 `plugin.yml`，避免菜单占位符被 Maven 改写
- 配置 Paper 26.2 全部 88 种刷怪蛋；五种特殊或高风险实体默认关闭
- Vault 或经济插件缺失时插件继续运行，菜单可浏览，所有经济交易被禁止
- 禁用分类不会再通过搜索结果暴露商品
- `/ds` 经济入账失败时恢复背包物品，不消耗出售额度

## 配置边界

- `SellShopconfig.yml`：收购规则、菜单和 1477 项商品价格
- `shopconfig.yml`：购买菜单、分类、覆盖价格、刷怪蛋、消息和日志

> **升级注意：** 插件不会覆盖服务器中已经存在的 `SellShopconfig.yml`。升级现有服务器时需要同步本次配置内容，否则新增商品和价格不会生效。

刷怪蛋统一配置在 `shopconfig.yml`，不进入随机收购池。
